# IP-Subnet-Validator

本项目是一个功能完备的IP地址管理与分析工具，深度集成了IP地址合法性校验与子网划分判断等核心功能。项目采用高内聚、低耦合的模块化架构设计，不仅实现了基础的IP格式规范化检查，更提供了涵盖网络掩码计算、广播地址解析、主机范围界定等在内的丰富网络计算能力，为网络规划与运维提供精准、可靠的技术支持。

## 项目结构

```
IP-Subnet-Validator/
├── main.py              # 主程序入口
├── ip_validator.py      # IP地址验证模块
├── ip_utils.py          # IP地址工具函数模块
├── network_calculator.py # 网络计算模块
├── subnet_divider.py    # 子网划分模块
├── display_formatter.py # 显示格式化模块
└── README.md           # 项目说明文档
```

## 模块说明

### 1. main.py
主程序入口，整合所有模块，提供用户交互界面。

### 2. ip_validator.py
IP地址验证模块，负责：
- 验证点分十进制+子网掩码格式
- 验证CIDR格式
- 检测各种IP地址输入错误

### 3. ip_utils.py
IP地址工具函数模块，提供：
- IP地址与整数的相互转换
- CIDR与子网掩码的相互转换
- 网络地址、广播地址计算
- 主机范围计算
- 子网容量计算
- IP地址类别判断
- IP地址类型判断（私有/公网/回环/组播/保留）

### 4. network_calculator.py
网络计算模块，负责：
- 计算完整的网络信息
- 获取子网详细信息
- 判断IP地址是否在子网中
- 获取IP地址在子网中的位置

### 5. subnet_divider.py
子网划分模块，提供：
- 固定长度子网划分
- VLSM（可变长子网掩码）划分
- 超网计算
- 最优子网查找

### 6. display_formatter.py
显示格式化模块，负责：
- 格式化输出网络信息
- 格式化输出子网列表
- 格式化输出VLSM结果
- 格式化输出超网信息
- 美化显示各种结果

## 功能特性

### 1. **IP地址验证**
   - 支持点分十进制+子网掩码形式：`192.168.1.1 255.255.255.0`
   - 支持CIDR形式：`192.168.1.1/24`
   - 检测输入不完整
   - 检测连续的点号
   - 检测非法字符
   - 检测点的数量是否正确
   - 检测字段数字是否在0-255范围内
   - 检测子网掩码的有效性

### 2. **网络信息计算**
   - 子网容量（总主机数和可用主机数）
   - 子网地址（网络地址）
   - 广播地址
   - 首个和最后主机地址
   - IP地址类别（A/B/C/D/E类）
   - IP地址类型（私有/公网/回环/组播/保留）
   - 二进制表示

### 3. **子网判断**
   - 判断两个IP地址是否在同一子网
   - 显示IP地址在子网中的位置

### 4. **子网划分**
   - 固定长度子网划分
   - 自动计算所有子网信息

### 5. **VLSM划分**
   - 根据不同子网需求进行可变长度划分
   - 优化地址空间利用率
   - 显示地址利用率

### 6. **超网计算**
   - 计算多个网络的超网
   - 显示超网详细信息

### 7. **最优子网查找**
   - 根据主机需求推荐最优子网
   - 显示地址浪费情况
